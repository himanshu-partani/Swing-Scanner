# Swing Scanner

A Python-based stock swing scanner built while learning Python, Pandas, and the Yahoo Finance API.

## Features

- Download live market data
- Calculate 20-Day Moving Average
- Calculate 50-Day Moving Average
- Compare price with moving averages
- Scan multiple stocks
- Export results to CSV

## Technologies

- Python
- Pandas
- yfinance
